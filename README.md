# MYGPT — AI Chatbot 🤖

A full-stack AI chatbot application built with the MERN stack, integrated with OpenAI GPT API.

## 🚀 Live Demo
🌐 https://my-gpt-chatbot-like-chat-gpt.vercel.app/

## ✨ Features
- AI-powered responses using OpenAI GPT API
- JWT-based user authentication
- Chat history stored in MongoDB
- Responsive UI built with React.js

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT
- **AI:** OpenAI GPT API

## ⚙️ Setup
1. Clone the repo
2. Add your OpenAI API key in `.env`
3. Run `npm install` in both client and server
4. Run `npm start`
